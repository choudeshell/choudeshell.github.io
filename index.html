Tattle - Yet Another Entity Framework Event Exposer

May 3, 2014 · 4 minute read
efc#

Debugging Entity Framework generated SQL is tough at times. The sometimes-elegant SQL is filled with parameters in every facet. One of the ways to view that SQL is to call .ToString() on any IQueryable. If you want to print the SQL out to the debug console in Visual Studio or even log the SQL – you’ll have to create a wrapped provider. Microsoft provides a logging and cache provider example. Based off the idea of this example, I created a wrapped provider of my own that should simplify tracing and logging of the generate SQL.
